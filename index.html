Un jeu Jumping développé par Terenceithaque à l'aide de la bibliothèque Pygame pour Python. Réalisez le meilleur score possible en attrapant les balles sans être écrasé(e) par les voitures !

Quelques infos avant de télécharger le jeu

- Le jeu a été développé avec Python 3, mais a ensuite été compilé pour fonctionner nativement sous le système d'exploitation Windows, sans l'aide de Python. Cela veut dire qu'il ne pourra pas fonctionner sur un ordinateur équipé d'un autre système d'exploitation, tel que Linux ou macOS.

- Une fois décompressé, le dossier du jeu (nommé "main") prendra environ 85 Mo d'espace sur votre disque.

- Vous trouverez au sein de ce même dossier l'exécutable du jeu, nommé "main.exe".

- Si lorsque vous double-cliquez sur l'exécutable, Windows vous affiche une boîte de dialogue "Windows a protégé votre ordinateur", ne vous inquiétez pas, il s'agit juste d'une mesure de prévention et l'exécutable du jeu ne contient aucune menace pour votre ordinateur. Cliquez donc sur "Exécuter quand même" si cette boîte de dialogue s'affiche pour lancer le jeu.

Télécharger l'archive du jeu

main.zip (à décompresser avec un logiciel tel que WinZip ou 7zip pour obtenir le dossier du jeu prêt à être exécuté).
